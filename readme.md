# MorseCode

A user-friendly Morse Code translation tool. Simply enter your message in either English or Morse Code and have it translated!

## Plans

- Morse Code memorization guide
- Morse Code playground website
- CLI Tool for quick translation

## Development

1. Install NPM packages: `npm install`
2. Start compiling TailwindCSS: `npm run watch:css`
3. Start compiling TypeScript: `npm run watch:ts`
